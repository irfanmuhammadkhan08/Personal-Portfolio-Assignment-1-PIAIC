# Accessibility Checker

A skill that helps analyze and improve the accessibility of web content to ensure it meets WCAG guidelines.

## Description

This skill assists in evaluating and enhancing the accessibility of your portfolio website by identifying potential barriers and providing solutions to make your content more inclusive for users with disabilities. It focuses on meeting WCAG 2.1 AA standards and best practices for web accessibility.

## Accessibility Knowledge

The skill understands and evaluates:
- WCAG 2.1 AA guidelines and success criteria
- Semantic HTML structure and proper heading hierarchy
- ARIA attributes and roles for enhanced accessibility
- Color contrast ratios and visual presentation
- Keyboard navigation and focus management
- Screen reader compatibility
- Alternative text for images and media
- Form accessibility and labeling
- Cognitive accessibility considerations

## Trigger Phrases

- "check accessibility"
- "accessibility audit"
- "WCAG compliance"
- "analyze for accessibility"
- "accessibility issues"
- "make accessible"
- "accessibility review"
- "test for ADA compliance"

## Parameters

None required.

## Implementation

When invoked, this skill will:
1. Analyze the provided HTML/CSS code or website content for accessibility issues
2. Identify problems with color contrast, semantic structure, and ARIA usage
3. Check for proper alternative text on images and media
4. Evaluate form accessibility and labeling
5. Assess keyboard navigation and focus management
6. Provide specific remediation suggestions for each identified issue
7. Generate accessible code alternatives where needed
8. Offer guidance on implementing accessibility improvements
9. Verify compliance with WCAG 2.1 AA standards
