# SEO Optimizer

A skill that helps optimize web content for search engines and improves overall site visibility.

## Description

This skill assists in optimizing your portfolio website for search engines by analyzing content, suggesting improvements, and implementing best practices for SEO. It focuses on technical SEO, content optimization, and on-page factors to improve search rankings.

## SEO Knowledge

The skill understands and applies:
- Technical SEO (meta tags, structured data, sitemaps, robots.txt)
- On-page optimization (title tags, meta descriptions, headers, keyword placement)
- Content optimization (readability, keyword density, internal linking)
- Performance factors (page speed, mobile-friendliness, Core Web Vitals)
- Accessibility considerations that impact SEO
- Local SEO basics

## Trigger Phrases

- "optimize for SEO"
- "improve search ranking"
- "analyze SEO"
- "SEO audit"
- "meta tags for"
- "structured data"
- "optimize content"
- "SEO best practices"

## Parameters

None required.

## Implementation

When invoked, this skill will:
1. Analyze the current page or content for SEO issues
2. Suggest improvements for title tags, meta descriptions, and headers
3. Recommend keyword optimization strategies
4. Identify technical SEO improvements (loading speed, mobile optimization)
5. Suggest structured data implementations
6. Provide recommendations for internal linking
7. Generate optimized meta tags and structured data snippets
8. Offer content improvement suggestions for better search visibility
