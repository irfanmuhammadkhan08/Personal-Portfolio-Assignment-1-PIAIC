# Playwright Testing

A skill that helps test your website visually using Playwright for automated browser testing.

## Description

This skill assists in creating and running Playwright tests to visually verify your website's appearance and functionality. It focuses on taking screenshots, checking responsive design across different viewport sizes, and verifying UI elements work as expected.

## Testing Knowledge

The skill understands and applies:
- Common viewport sizes for responsive testing (mobile: 375x667, tablet: 768x1024, desktop: 1920x1080)
- Visual regression testing techniques
- Cross-browser compatibility testing
- Responsive design verification across devices
- UI element interaction and validation
- Screenshot comparison methodologies
- Headless and headed browser testing
- Performance and loading state verification

## Trigger Phrases

- "test with Playwright"
- "visual testing"
- "responsive design test"
- "screenshot test"
- "UI verification"
- "cross-browser test"
- "viewport testing"
- "visual regression"

## Parameters

None required.

## Implementation

When invoked, this skill will:
1. Generate Playwright test scripts for visual verification
2. Create tests that take screenshots at different viewport sizes
3. Implement responsive design checks across common device sizes
4. Verify UI elements are properly displayed and functional
5. Generate visual regression tests to compare current vs expected appearance
6. Create cross-browser compatibility tests
7. Implement loading state and animation verification
8. Provide test reports and screenshot comparisons
9. Suggest improvements to test coverage and reliability
