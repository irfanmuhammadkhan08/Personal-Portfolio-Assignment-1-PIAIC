# UI/UX Designer

A skill that helps design beautiful UI components following established design principles and systems.

## Description

This skill assists in creating beautiful, accessible UI components by leveraging knowledge of your design system including color palette, spacing system, and accessibility guidelines. The skill generates clean HTML/CSS code that follows your established design system.

## Design Knowledge

The skill understands and applies:
- Your color palette (primary: blue #3498db, secondary: purple #9b59b6, neutral grayscale)
- Spacing scale based on 8px grid system (8px, 16px, 24px, 32px, 48px, 64px)
- Typography scale for proper hierarchy
- Accessibility standards (semantic HTML, ARIA attributes, proper contrast ratios)
- Component patterns (buttons, cards, navigation, forms)

## Trigger Phrases

- "design a component"
- "create a UI element"
- "make a button/card/form"
- "help with UI design"
- "create accessible component"
- "design following system"
- "build UI component"

## Parameters

None required.

## Implementation

When invoked, this skill will:
1. Identify what UI component the user wants to create
2. Apply your design system principles (colors, spacing, typography)
3. Generate semantic HTML with proper accessibility attributes
4. Create CSS that follows your design system
5. Ensure the component is responsive and accessible
6. Provide clean, well-commented code that integrates with your existing codebase
